# ADR-008: Teto de custo de R$20/mês

## Status

Accepted

## Contexto

O custo é requisito arquitetural e restringe tecnologias e padrões operacionais.

## Decisão

Toda decisão técnica deve justificar permanência abaixo de `R$20/mês`.

## Consequências

- priorização de serviços pay-per-use
- exclusão de serviços com custo fixo nesta fase
- pressão constante por simplicidade arquitetural
